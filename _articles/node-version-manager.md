### 安装

- [nvm-sh/nvm](https://github.com/nvm-sh/nvm) Linux & Mac
- [coreybutler/nvm-windows](https://github.com/coreybutler/nvm-windows) Windows

### 常用命令

- `nvm list` 展示当前已安装版本
- `nvm install <version>` 安装指定版本
- `nvm use <version>` 使用指定版本
